# Governance and Accountability L4 Evaluation

## Project name

This project evaluates whether AI providers have established codes of conduct, defined roles, and training programs to support responsible AI governance.[web:25][web:28]

## Prerequisites

- Python 3.9+ installed.  
- Required Python packages installed (for example via `pip install -r requirements.txt`).[web:44]  
- An API key for your chosen LLM provider.

## Project structure

- Each folder containing the governance-related PDFs and generated outputs.  
- `script.py` – main script that evaluates the documents against L4 governance criteria.  
- `outputs/` – evaluation results for each input document.

Update the folder names above if your actual repo layout is slightly different.[web:70]
